* Vision

Emacs core C を Elisp で再実装、NeLisp runtime 上で動作させる pure
Elisp Emacs を build する long-term project。

* Relationship to NeLisp

- Layer 1 (NeLisp core runtime) = =/Notes/dev/nelisp/= (= 既存、本 repo
  触らない)
- Layer 2 (extension package) = 一部既 SHIPPED 済 (=
  =nelisp-emacs-compat= / =nelisp-process= / =nelisp-eventloop-multiplex=
  / =nelisp-regex= / =nelisp-coding= / =nelisp-sqlite= / =nelisp-base64=
  / =nelisp-secure-hash= / =nelisp-json= / =nelisp-filenotify= 等)、本
  repo は **残 Layer 2 + Layer 3 (display backend)** を担当
- Layer 3 (display backend) = TUI primary、GTK / Cocoa / Win32 / wgpu は
  community

* Initial Scope (Phase 1)

per Doc 01 architecture LOCKED:
1. =src/emacs-buffer.el= = buffer.c port (= 既 nelisp-emacs-compat の上
   に乗る、buffer-local-variable / advanced text-property / undo full)
2. =src/emacs-window.el= = window.c port
3. =src/emacs-frame.el= = frame.c port (= display backend abstraction)
4. =src/emacs-keymap.el= = keymap.c port
5. =src/emacs-minibuffer.el= = minibuffer.c port
6. =src/emacs-tui-backend.el= = TUI display backend MVP (=
   ncurses-style、reference impl)
